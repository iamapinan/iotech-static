แนะนำเว็บไซต์จับคู่สี

Apinan Woratrakun
1 ปี ago

แนะนำเว็บจับคู่สีเพื่อนำไปใช้ในงานออกแบบต่างๆ เช่น ux ui web design

1. Color Adobe

เว็บสร้างชุดสีนี้เป็นของบริษัท Adobe ที่เรียกได้ว่าครบเครื่องมากๆ เพราะเราสามารถกำหนดได้ว่าต้องการชุดสีประเภทไหน
ซึ่งมีทั้งโทนเดียวกัน โทนที่ตรงข้ามกับสีหลัก โทน 3 สีที่ใกล้เคียงกันและอื่นๆ โดยรูปแบบจะเป็นกงล้อให้เราเลือกสีหลัก
เมื่อได้สีที่ต้องการก็เก็บไว้ในประวัติของเรา หรือถ้าเกิดขี้เกียจสร้างเองก็เลือกดูชุดสีที่มีคนสร้างไว้แล้วก็ได้
นอกจากนี้เรายัง import ไฟล์รูปภาพจากคอมพ์ของเราเพื่อเช็คดูโทนสีได้อีกด้วยนะ

2. Colour Lover

เว็บนี้เป็นแหล่งรวบรวมชุดสีที่เว็บไซต์ต่างๆ ใช้งานเอาไว้เยอะมากๆ โดยหน้าเว็บได้จัดประเภทของเว็บไซต์เอาไว้เป็นหมวดๆ
ทั้ง Home, Fashion, Business และอื่นๆ เพื่อช่วยให้เราเลือกชุดสีที่เหมาะสมกับงานของเราได้สะดวกยิ่งขึ้น
เมื่อเลือกชุดสีที่ชอบได้แล้ว ก็สามารถคลิกเข้าไปดูรายละเอียดของสีนั้นๆ ได้อีกด้วยนะ

3. Coolors

เว็บนี้ต่างจากเว็บอื่นๆ ตรงที่ไม่ได้จับคู่สีให้เราอัตโนมัติ แต่เราสามารถกำหนดค่าของสีแต่ละสีด้วยตัวเองได้อย่างอิสระ
หากเราเลือกจนพอใจแล้ว ก็สามารถ export ชุดสีเป็นไฟล์ต่างๆ เพื่อนำไปใช้กับงานกราฟฟิของเราได้
และยังมีชุดสีสวยๆ ที่สร้างไว้แล้วสำหรับคนที่ขี้เกียจสร้างเอง

4. Paletton

เป็นเว็บที่ให้เราเลือกสีได้ในรูปแบบกงล้อคล้ายๆ กับ Color Adobe กำหนดได้อีกเช่นกันว่าจะเอาชุดสีประเภทไหน
ทั้งแบบโทนเดียวกัน โทนที่ตรงข้ามกับสีหลัก โทน 3 สีที่ใกล้เคียงกันและอื่นๆ
เมื่อได้ชุดสีที่ต้องการแล้ว ก็สามารถ Export ไปใช้กับงานกราฟฟิคต่างๆ ของเราได้ทันที
