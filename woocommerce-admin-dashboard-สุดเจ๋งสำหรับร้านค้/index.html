WooCommerce Admin Dashboard สุดเจ๋งสำหรับร้านค้า

  • Apinan Woratrakun
  • 23/06/2019
  • Open Source / Web Development

เปิดให้ดาวน์โหลดแล้ว Woocommerce Admin ตัวเสริมที่รอมาชาติครึ่ง

สำหรับคนที่ทำเว็บร้านค้าออนไลน์ที่ใช้ WordPress และ Woocommerce เป็นระบบหลักนั้นเมื่อก่อนจะติดปัญหาอย่างหนึ่งคือระบบรายงานสถิติ Dashboard หรือการวิเคราะห์ผลกำไรต่างๆต้องใช้ปลั๊กอินอื่นมาเสริมซึ่งเป็นของผู้พัฒนาเจ้าอื่น โดยในส่วนของ Woocommerce นั้นมีเพียงหน้ารายงานอันเดียวที่ดูได้แค่ยอดขายแถมยังทำเอาผู้ใช้สับสนในหลายๆครั้ง

และแล้วก็มีวันนี้กับการเปิดตัว Woocommerce Admin ซึ่งเป็นเหมือนระบบรายงานและ Dashboard สุดเจ๋งที่รวมสิ่งต่างๆที่คนทำร้านค้าออนไลน์ต้องการและใช้งานง่ายมากขึ้น เรียกได้ว่า ถึงขนาดอาจจะไม่ต้องหนีไปใช้พวก Magento เลยก็ได้เพราะตอนนี้มีระบบวิเคราะห์ต่างๆให้หมดแล้วนั้นเอง มาดูกันว่ามีอะไรเด่นกันบ้าง

New Reports for WooCommerce (เมนู Analytics) ส่วนรายงานผลและยอดขายใหม่ทั้งหมดประกอบด้วยการรายงาน

  • Revenue Report แสดงรายได้ ยอดขาย กำไรของร้านค้า และภาพรวมอื่นๆ
  • Orders Report แสดงยอดสิ่งซื้อ
  • Products Report แสดงรายงานสินค้าที่ขาย
  • Categories Report แสดงรายงานตามหมวดหมู่สินค้า
  • Coupons Report แสดงรายงานคูปองที่ถูกใช้
  • Taxes Report แสดงรายงานภาษีและค่าธรรมเนียม
  • Downloads Report แสดงรายงานสินค้าที่ดาวน์โหลด
  • Stock Report แสดงรายงานสต๊อกสินค้า
  • Customers Report แสดงรายงานเกี่ยวกับลูกค้า

จะเห็นว่าจุดที่เพิ่มขึ้นมาอย่างเห็นได้ชัดเลยคือเมนูรายงานที่รวมรายงานทุกๆด้านที่เราควรมีไว้ที่เดียวเลย ซึ่งในหน้ารายงานเหล่านั้น เราสามารถปรับเปลี่ยนขอบเขตของรายงานตามที่เราต้องการได้ด้วยโดยเลือก Date Range ด้านบน

นอกจากการรายงานแล้วยังมีจุดที่เปลี่ยนแปลงเพิ่มขึ้นเช่น เมนู Dashboard ที่จะมาอยู่ก่อนเมนู Orders เดิม

และยังมีแถบนำทางใหม่เพิ่มเข้ามาในส่วนของระบบหลังบ้านด้วย

ซึ่งด้านซ้ายจะเป็น breadcrumb ที่บอกว่าตอนนี้เราอยู่ในส่วนไหนของระบบ และด้านขวาจะเป็นเมนูแจ้งเตือนกิจกรรม ซึ่งประกอบด้วย

  • Inbox กล่องข้อความจากระบบ
  • Orders เพื่อดูการแจ้งเตือนออร์เดอร์ใหม่ ซึ่งเราสามารถคลิกดำเนินการต่างๆได้เลยโดยไม่ต้องเปิดหน้าออร์เดอร์
  • Stock เพื่อดูการแจ้งเตือนสต๊อกสินค้า
  • Reviews เพื่อดูการแจ้งเตือนการรีวิวใหม่
  • Notices เพื่อดูข้อความเตือนจากระบบเช่นการอัพเดทการทำงานเบื้องหลังบางอย่าง

ปัจจุบันเป็นเวอร์ชั่น 0.13.2 แล้วและมียอดดาวน์โหลดแล้ว 100,000 กว่า หากสนใจอยากลองใช้งานก็สามารถดาวน์โหลดที่ Plugin Directory ของท่านเองได้เลย

หรือใครใจร้อนอยากลอง Features ใหม่ๆที่อยู่ในช่วงพัฒนาด้วยก็ไปดาวน์โหลดมาใช้เองได้เลย

https://github.com/woocommerce/woocommerce-admin/tree/v0.13.2

ซึ่งไม่รับรองความเสถียรนะครับ ด้านล่างนี้เป็นบทความต้นทางจากทีมผู้พัฒนา Woocommerce และลิ้งค์สำหรับดาวน์โหลดปลั๊กอินนี้นะครับ
