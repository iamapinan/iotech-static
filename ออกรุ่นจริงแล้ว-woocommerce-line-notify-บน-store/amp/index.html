Apinan Woratrakun
4 เดือน ago

Woocommerce Line Notify รุ่นสมบูรณ์ 1.0.9
โหลดกันได้แล้วนะครับ วิธีทำ
1. ลบตัวเก่าทิ้งไปเลย ค่าเดิมไม่หายไปไหน
2. เข้าไปที่ เพิ่มปลั๊กอิน แล้วค้นหา iamapinan ตรงข้างหน้าช่องค้นหาให้เลือกเป็น ผู้เขียน หรือ author เพราะ plugin ใหม่หาเจอยากอยู่ท้ายเลย แบบนี้ง่ายกว่า
3. หรือดาวน์โหลดได้ที่ https://wordpress.org/plugins/woo-line-notify

ฝากติดตามผลงานด้วยนะครับ กด Link iOTech

รุ่น 1.0.7 – 1.0.9 มีอะไรปรับปรุงบ้าง

  • Change setting design.
  • Add debug function.
  • Update code and folder structure.
  • Change plugin slug to match with Plugin directory.
  • Fixed many warning.
  • Remove banner from setting page.
  • Add icon to setting page.
  • Make all text translatable.
  • Full support Thai and English language.
  • Change token field type to password.
