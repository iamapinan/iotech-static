6 Tools for web & mobile app design | รวมเครื่องมือออกแบบเว็บและแอพลิเคชั่น

  • Apinan Woratrakun
  • 22/06/2018
  • Creative and Design

Marvel app

เว็บแอพลิเคชั่นที่ช่วยในการออกแบบ Workflow ของ UX UI เพื่อให้เห็นภาพรวมของงานรวมทั้งยังสามารถกลับมาแก้ไข้ให้แอพลิเคชั่นที่เราออกแบบสมบูรณ์มากยิ่งขึ้นก่อนนำไปพัฒนาเป็นโปรแกรมจริง เครื่องมือนี้จะช่วยอำนวยความสะดวกมากๆ เพราะมีหลายอุปกรณ์ให้พรีวิวดูได้เพื่อความสมจริง

link : https://marvelapp.com/

InVision

อีกเครื่องมือหนึ่งที่ช่วยออกแบบ Prototyping ที่น่าสนใจไม่แพ้กันที่ช่วยในการออกแบบ Workflow มีแอคชั่นให้เล่น จุดเด่นคือ Collaboration ที่สามารถทำงานร่วมกันในแต่ละโปรเจคบนนี้ได้

link : https://www.invisionapp.com/

Google Fonts

ฟ้อนต์ฟรีจาก Google ที่ใช้ง่ายมากๆ ที่ให้เลือกทั้งภาษาไทยและภาษาอังกฤษ ช่วยลดปัญหาการใช้ฟ้อนต์และใช้งานง่าย เป็น Open source เบราเซอร์รองรับเยอะและยังสามารถนำไปใช้กับ css โดยไม่ต้องไปยุ่งกับโค้ดส่วนอื่นๆ

link : https://fonts.google.com/

Wordmark

เว็บดูฟ้อนต์แบบออนไลน์ที่เป็นเครื่องมือช่วยในงานออกแบบที่ช่วยให้ฟ้อนต์ต่างๆ ที่อยู่ในคอมเราก็จะปรากฏตัวพรึ่บขึนมาทันที!

link : https://wordmark.it/ 

Khroma

แนะนำเว็บไซต์หาคู่สีที่มี AI มาช่วยค้นหา

link : http://khroma.co/

Webgradients

ถ้าใครอยากออกแบบเว็บไซต์หรือแอพลิเคชั่นโดยใช้การไล่สีเพื่อให้งานของคุณดูทันสมัยมากขึ้น แต่อาจจับคู่สีไม่เก่ง แนะนำเว็บนี้เลย

link : https://webgradients.com/

 

 

 

 

 

 

 

 

 

 

 

 

 

 

ขอให้สนุกกับการเริ่มต้นออกแบบเว็บไซต์และแอพลิเคชั่นจากเครื่องมือง่ายๆ ที่แนะนำ 🙂
