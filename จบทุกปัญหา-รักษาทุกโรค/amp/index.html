นี่คือคำอวยตัวเองของสุดยอดปลั๊กอินเปลี่ยนโลกที่ผมอยากจะแนะนำเพื่อนๆในครั้งนี้ครับ

ก่อนอื่นต้องขอบอกก่อนว่า หลายวันก่อนตอนที่ผมกำลังทำ Research เพื่อหาปลั๊กอินที่ดีๆมาแนะนำคนที่จะเข้ามาเรียนคอร์สของผมซึ่งที่ผ่านไปก็คือ Introduction to WordPress for Startup/SME ผมได้ไปเจอ Plugin ตัวนึงที่มีผู้ใช้ไม่เยอะ แต่โม้ไว้ว่า ปลั๊กอินเดียวแก้ปัญหาของ WordPress ได้เกือบหมด

One plugin to solve WordPress’s biggest problems.

ผมเลยไม่เชื่อ ก็เลยลองไปอ่านดูรายละเอียด และโหลดมาลองใช้ ผลก็คือ

ปลั๊กอินตัวนี้มีชื่อว่า WP2Static แค่ฟังชื่อก็พอเดาออกละ แหมน่าสนใจ มาดูว่ามันทำอะไรให้เราบ้าง

ให้เราแปลงเว็บไซต์ของเราที่ทำด้วย WordPress ให้กลายเป็น HTML ธรรมดาๆ ถึงกับต้องแปลกใจและอุทานว่า OMG! นี่มัน Killer feature ชัดๆ โดยเราจะสามารถแปลงเว็บเราทั้งเว็บที่ออกแบบและพัฒนาบนคราบน้ำตาของเราเองมาอย่างยาวนานนั้น ให้กลายเป็น HTML ที่มีข้อดีมากมายอาทิเช่น

  1. เอาไปวางไว้ที่ไหนก็ได้ ไม่ว่าจะเป็น Firebase hosting, Github page, FTP, S3 ทุกๆที่ๆคุณอยากจะไป ฟรีๆทั้งนั้นที่เห็นๆ
  2. เอาไปใส่ Flash (drive) express ส่งด่วนไปให้ใครเอาไปดูโดยไม่มี Server ก็ได้เพราะมันรันใน Web browser
  3. เร็วมากๆๆๆๆๆๆ … เพราะมันทำงานบน client
  4. ลืมเรื่อง Backend Caching ไปได้เลยเพราะต่อไปนี่มันอยู่บน Github page ได้แล้ว

ปลอดภัยมากขึ้น เพราะเราไม่ต้องมี Server หลังบ้าน ไม่ต้องมี WordPress ทำงานก็ได้มีแต่ HTML เพียวๆ จะแฮกเอาอะไร แถมยังสามารถใส่ Basic Authen เข้าไปได้ด้วย ทำให้เราสามารถป้องกันการเข้าถึงได้ด้วย username/password ก่อนเข้าเว็บได้ด้วย

ทำ Auto deploy ได้ด้วย เนื่องจากบางครั้งเว็บเราต้องมีการแก้ไขเช่นเพิ่มบทความใหม่ หรือปรับแต่งเว็บตามเทศกาล จึงทำให้เว็บเรามีการเปลี่ยนแปลง ซึ่งปลั๊กอินสามารถทำ Automate Deploy ในบางที่ได้เช่น Github page, S3, Gitlab, etc..

กรอกโค้ดขยะออกได้บ้าง โดยตัวปลั๊กอินสามารถให้เราตั้งค่าว่าจะให้ลบพวกโค้ด comments ออกลบ meta tag ออก ลบโค้ดที่ไม่จำเป็นออกได้ด้วย

เปลี่ยน path ของไฟล์ทำให้เหมือนว่าไม่ได้ใช้ WordPress ช่วยให้เว็บที่เรา Export เป็น Static ไปนั้นเหมือนถูกสร้างแบบเขียนโค้ดเอง โชวความดิบของเราให้โลกรู้ไปอีก และที่สำคัญมันยังเปลี่ยนโดเมนได้ด้วยนะ

มี Hooks ให้เราเรียกทำงานกันแบบอัตโนมัติ ใครที่ชอบทำเว็บแล้วให้มัน Build Test Deploy น่าจะถูกใจสิ่งนี้ เพราะเราสามารถเรียก Hook เพื่อสั่งให้มัน export และส่งไฟล์ที่ได้ไป deploy ใน firebase หรืออะไรก็ตามที่ต้องการได้

มีระบบแจ้งเตือนเมื่อทำงานเสร็จ ต่อจากเรื่อง hook พอมันทำงานเสร็จแล้วมันยังแจ้งเตือนเราได้หลายทางทั้ง Desktop notification หรืออีเมล์ที่เรากำหนด ช่วยให้เราไม่ต้องมานั่งรอมันทำงาน

บังคับให้ไปที่ใหม่ คนที่เข้าเว็บเดิมที่เป็น WordPress ให้วิ่งไปยังเว็บใหม่ที่เป็น Static ได้เลย

ยังมีคุณสมบัติอื่นๆอีกเยอะเลย แต่จากที่กล่าวมาก็พอจะทำให้นึกภาพออกว่า ปลั๊กอินตัวนี้เปลี่ยนชีวิตเราได้จริงๆ แต่ก็พอจะมีข้อเสียอยู่บ้างก็ตรงที่ใช้เวลานานเหมือนกันกว่าจะทำเสร็จ ถือซะว่าพักดื่มกาแฟสักแก้ว และอีกเรื่องคือเว็บที่ต้องใช้ข้อมูลที่เป็น Dynamic อย่างเว็บ eCommerce ดูอาจจะไม่เหมาะสักเท่าไหร่ทุกอย่างย่อมมีข้อจำกัด แต่เมื่อมองภาพรวมที่เกิดขึ้นคือ เราสามารถทำเว็บด้วยเครื่องมือที่สุดยอดมากอย่าง WordPress จนได้เว็บสวยๆ เริ่ดๆ มาแล้วเราก็ไม่ต้องห่วงเรื่องอะไรอีก แม้กระทั้งเรื่องความเร็ว ความปลอดภัย ปัญหาจุกจิกของ Server และที่สำคัญคือเซิฟเวอร์เราจะไม่ร่วงเมื่อมีคนใช้เยอะๆอีกต่อไปเพราะเราใช้ Cloud ฟรีของคนอื่น อิอิ ^^

ซึ่งต้องบอกเลยว่าถ้าเอาไปใช้จริงๆ ซึ่งเดี่ยวผมนี่แหละจะลองกับเว็บบริษัทฯ มันจะเป็นอะไรที่สุดยอดมาก นี่แหละปลั๊กอินเปลี่ยนโลก ณ วันนี้ผมเห็นคนใช้จริงเพียงหลักหมื่นกว่าๆ แต่เชื่อว่าไม่นานมันต้องมีคนใช้เยอะอย่างก้าวกระโด่ดแน่นอน ถ้ายังคงรักษาความดีงามและการพัฒนาที่ต่อเนื่อง ที่สำคัญคือฟรี เอาล่ะไม่ใช่ของตัวเองแต่ดีอย่างนี้ก็ช่วยขายฟรี เพื่อเป็นกำลังใจให้คนพัฒนาครับ โหลดฟรีได้ที่นี่เลย
