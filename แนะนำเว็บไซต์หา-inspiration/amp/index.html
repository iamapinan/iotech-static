แนะนำเว็บไซต์หา Inspiration

Apinan Woratrakun
1 ปี ago

มัดรวมเว็บไซต์เด็ดๆ ที่เอาไว้หา Inspiration ในการออกแบบ UX UI และ Web design นอกจากจะได้ดูงานที่เราสนใจแล้วเพื่อนำไปพัฒนางานออกแบบต่อ ไปและยังเป็นการอัพเดคเทรนด์ในช่วงเวลาต่างๆด้วย

1. DRIBBBLE

เว็บไซต์นี้เปิดโอกาสให้ดีไซเนอร์จากทั่วทุกมุมโลกมาลงงานดีไซน์ของตัวเองโดยแต่ละงานก็สวยๆ ทั้งนั้น เพราะไม่ใช่ใครก็ลงผลงานในเว็บนี้ได้ แต่ต้องมี Invite จากดีไซเนอร์ที่อยู่ใน Dribbble อยู่แล้ว

2. BEHANCE

เว็บไซต์ Behance ก็เป็นเว็บชื่อดังในวงการดีไซเนอร์ต่างประเทศ ซึ่งในเว็บไซต์นี้จะมีงานดีไซน์ทุกแขนงให้ดู ไม่ว่าจะเป็น Print Design, Web Design, Package Design, Branding, Typography และอื่น ๆ อีกมากมาย

3. PINTEREST

เป็นเว็บไซต์ที่เอาไว้หา Inspiration ได้ดีมาก ค้นอะไรก็เจอมีการแสดงผลเป็น Grid ทำให้สามารถดูงานได้เยอะขึ้น

4. AWWWARDS

เว็บไซต์ Awwwards จะรวมเว็บสวย ๆ จากทั่วทุกมุมโลก ซึ่งแต่ละวันมีเว็บไซต์ส่งเข้ามาเข้ามามากมาย เว็บไซต์ไหนได้คะแนนโหวตเยอะที่สุดก็จะถูกเลือกเป็น Site of the day

5. PTTRNS

อยากดูเฉพาะดีไซน์ของหน้าจอแบบไหนก็กด Filter ได้ โดยเว็บไซต์นี้จะมีงานดีไซน์เป็นภาพนิ่งให้ดูเท่านั้น

 

เว็บไซต์อ้างอิง : https://www.designil.com/5-inspiration-website-designer.html
